## Ten-Minute Review Path

1. Read the board-level reporting pack.
2. Inspect the governance operating model and ownership matrix.
3. Review the top risks and one control-mapping example.
4. Inspect the evidence catalogue and policy-pack examples.
5. Review the prioritised governance roadmap.
