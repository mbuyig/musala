# Weekly Archive

Keep only the active week in [../now.md](../now.md). At the end of the week,
copy [template.md](template.md) to a dated file using the Monday start date:

```text
tracking/weekly/YYYY-MM-DD.md
```

For example:

```text
tracking/weekly/2026-06-01.md
```

Then move the completed actions, notes and unfinished work from
[../now.md](../now.md) into that dated file. Start the next week with a clean
active checklist.
